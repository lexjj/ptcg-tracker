⚙ GitHub Sync Setup

Your expense data is saved to a data.json file in a GitHub repo. Any device that opens this app will load the same data.

To create a token: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → New token.
Required permission: Contents → Read and Write for this repo only.
The token is stored only in this browser's localStorage — never sent anywhere except GitHub's API.
